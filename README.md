# ☕ Coffeexhop

Sitio web moderno para una **cafetería**, desarrollado con **Astro** y **Tailwind CSS**.  
Este proyecto combina diseño, rendimiento y buenas prácticas de desarrollo para crear una experiencia atractiva para los usuarios.

---

## 🚀 Características

- 🌐 **Astro + Tailwind CSS** → Diseño moderno, responsive y optimizado.
- 📰 **Blog dinámico** con:
  - **SSR (Server-Side Rendering)**
  - **SSG (Static Site Generation)**
- 🔗 **Routing dinámico** para manejar entradas de blog u otras secciones basadas en estructuras de datos.
- ☁️ **Integración con WordPress** mediante la **WP REST API** para obtener:
  - Títulos
  - Contenido
  - Imágenes
  - Campos personalizados
- 📦 **Integración de librerías desde NPM**:
  - [Leaflet](https://leafletjs.com/) → Mapas interactivos
  - [Notyf](https://github.com/caroso1222/notyf) → Notificaciones
  - [PhotoSwipe](https://photoswipe.com/) → Galerías de imágenes
- 📬 **Formularios de contacto** con soporte para **Actions**.

## 📚 Aprendizajes Clave

En este proyecto aprenderás:
- A trabajar con **Astro** y **Tailwind** para crear sitios web modernos.
- Cómo consumir datos desde la **WP REST API** de WordPress.
- A implementar **SSR y SSG** en un mismo proyecto.
- Uso de **routing dinámico** para entradas de blog.
- Integración de librerías JS desde NPM.
- Manejo de **formularios de contacto** y acciones del lado del servidor.
